# 리스트 컴프리헨션
- 리스트 안에 for문을 내장한 형태.
```python
# 깡 코드
a = [1,2,3,4]
result = []
for num in a:
    result.append(num*3)
print(result)
```
```python
# 리스트 컴프리헨션 기본형
a = [1,2,3,4]
result = [num * 3 for num in a]
```
```python
# 리스트 컴프리헨션, if 문 추가
a = [1,2,3,4]
result = [num * 3 for num in a if num % 2 == 0]
```

# for 를 활용하는 메서드들
## enumerate
- 리스트의 내부를 index 와 함께 묶어준다.
```python
fruits = ['apple', 'banana', 'orange']
for i, fruit in enumerate(fruits):
    print(f" {i}: {fruit} ")
# 0: apple
# 1: banana
# 2: orange
```
- 시작 범위 조절도 가능
```python
fruits = ['apple', 'banana', 'orange']
for i, fruit in enumerate(fruits,1): #1부터 시작
    print(f" {i}: {fruit} ")
# 1: apple
# 2: banana
# 3: orange
```

## zip
- 리스트 두개를 묶어준다.
- 서로의 인덱스가 같은 것끼리 짝지어 튜플화
```python
names = ['홍길동', '김철수', '이영희']
scores = [85,92,78]
for name, score in zip(names,scores):
    print(f"{name}: {score}점")
# 홍길동: 85점
# 김철수: 92점
# 이영희: 78점
```
