# 문자열 슬라이싱
- 기본적으로 파이썬은 문자열을 char 배열처럼 다룬다.
- 따라서 인덱스를 통해 문자열의 특정 글자에 접근 가능
```python
string = "Life is short, you need python"
letter = string[0]
print(letter) # L
```
- 슬라이싱 문법이란게 있다.
- 시작점 : 끝점을 정의해 해당 인덱스 범위의 것들만 뽑아내는 것.
- 이 때, 시작점은 포함 끝점은 포함하지 않는다.
- 다시 말해 이상:미만 문법으로 동작.
```python
string = "Life is short, you need python"
letters = string[0:4]
print(letters) # Life
```
# 문자열은 immutable
```python
# 여기 오타가 난 Python 문자열이 있다.
a = "Pithon"
# 이렇게 i를 y로 대입해 바꿀 수 있을 것 같지만,
a[1] = 'y'
# 위 코드는 오류가 난다. 문자열은 immutable 이기 때문.
# 올바른 방법은, 앞뒤를 슬라이싱하고 y를 끼워넣어 새로운 문자열을 만드는 것 뿐이다.
```
# 포매팅
- 클래식한 C 스타일의 방법
```python
# 자연수는 %d
string = "I eat %d apples" % 3
print(string) # I eat 3 apples

# 문자열은 %s
string = "I eat %s apples" % "five"
print(string) # I eat five apples

# 변수도 사용가능
num = 3
string = "I eat %d apples" % num
print(string) # I eat 3 apples

# 2개 이상 넣기는 튜플로
string = "I eat %d apples. so I sick for %s days" % (3, "three")
print(string) # I eat 3 apples

# 포매팅 연산자가 있는 문자열에서 퍼센트%를 표현할 때는 %%로 이스케이프가 필요하다.
```
## 정렬 형태로 포매팅
```python
## %s 가 문자열포매팅, %10s 는 전체 10자리 문자열을 만들고 문자열은 오른쪽정렬.
string = "%10s" % "hi"
# (공백8자)hi 가 출력된다. len이 고정된 문자열을 찍어내고 싶을 떄 유용.

## 왼쪽 정렬 : %-10s
string = "%-10sjane" % "hi"
# hi(공백8자)jane
```

## format 함수
```python
string = "I eat {0} apples".format(3)
# I eat 3 apples
string = "I eat {0} apples".format("five") #포맷 함수는 문자열도 그냥 박아버림
# I eat five apples
string = "I eat {0} apples. for {1} days".format("five", 3) #인자들을 0 1 순서에 맞게 박는다.
# I eat five apples for 3 days
string = "I eat {num} apples. for {day} days".format(num="five",day=3) #이름있는인자
# I eat five apples for 3 days

# 정렬
string = "{0:<10}".format("hi") # <는 왼쪽정렬
## hi(공백8자)
string = "{0:>10}".format("hi") # >는 오른쪽정렬
## (공백8자)hi
string = "{0:^10}".format("hi") # ^는 가운데정렬!!!
## (공백4자)hi(공백4자)
string = "{0:=^10}".format("hi") # 앞에 =를 두면 =패딩
## ====hi====
string = "{0:!>10}".format("hi") # 앞에 !를 두면 !패딩
## !!!!!!!!hi

# 소숫점 포매팅
y = 3.141592
string = "{0:0.4f}".format(y)
# 3.1415

y = 3.141592
string = "{0:10.4f}".format(y) # 총 len 을 10,소숫점을 4자리까지
# (공백 4칸)3.1415

# 중괄호 표현하기
string = "{{ 이러면 중괄호임 }}" # { 이러면 중괄호임 }

# f문자열은 거의 생활처럼 써왔으니 생략.
## 금액 콤마 표시나 알아두자
```
