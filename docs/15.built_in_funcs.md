# dir
- 객체가 가진 변수나 함수를 알려줌
```python

```

# filter(함수, 이터러블)
- 이터러블에 함수를 적용해, True 인 것만 골라 반환.
- 마지막에 필터의 반환값을 list() 하는 등의 형변환 요구
```python
list(filter(lambda x: x>0, [1,-3,2,0,-5,6]))
```
